# Third-party notices and provider register

Go module and npm lockfiles are the authoritative dependency inventories. The repository SBOM must be regenerated for the final source commit. No third-party cloud, AI, exchange or settlement provider is enabled by default.

Any provider adapter must record legal entity/service, official API or SDK, license, terms URL and version, jurisdiction, authentication method, rate limit, retention, data rights, health source, fallback behavior and outage state. Credentials remain in operator secret storage and never enter browser bundles, logs, evidence files or chat.

The current local capacity probe uses Node.js built-in `fetch` and ephemeral local provider fixtures. Those fixtures are isolated test evidence and are not production providers, capacity or revenue.
