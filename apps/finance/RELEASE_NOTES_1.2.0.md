# YNX Finance 1.2.0 — Testnet candidate

This release replaces the legacy local session boundary with canonical Wallet authorization, adds real-source activity coverage and source status, private notes, budget progress, monthly review, account deletion and stronger recovery/audit controls. The native UI now supports system light/dark themes, icon navigation, 12 locales including Arabic RTL, locale-aware YNXT/date formatting and explicit offline/unavailable states.

AI categorization, fee explanations and budget drafts remain consented review drafts with Apply/Reject/Delete controls. Finance cannot sign, transfer, trade, stake, freeze or custody assets, and it never invents fiat balances, APY, receipts or yields.

This is not a production release. The Android APK is locally test-signed. Central Wallet registry/deployment, installed Wallet approval, authorized Pay receipt smoke, iOS Simulator evidence, production signing and store review remain pending.
