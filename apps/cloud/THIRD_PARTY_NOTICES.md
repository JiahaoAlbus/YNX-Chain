# Third-party notices and provider register

The Android artifact contains dependencies resolved by the checked-in native lockfiles and Gradle build. `evidence/SBOM_REVIEW.md` records the current review boundary; it is not a complete production SBOM.

No production object-storage, CDN, KMS, antivirus, AI, billing, or support provider is approved. Before enabling one, record official product/API, license and terms, jurisdiction/data residency, authentication, retention and data rights, API version, rate limits, health endpoint, outage behavior, fallback, export/exit, and exact dependency license notices.

YNX metadata and authorization remain authoritative. A provider response never substitutes for Wallet identity, ACL, quota, audit, settlement, or chain evidence.
