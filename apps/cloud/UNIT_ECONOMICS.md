# Unit economics

No price, margin, revenue, burn, or subsidy is currently claimed. The local adapter has no defensible provider cost model.

Before charging users, meter storage byte-hours, ingress, egress, scan bytes, compute seconds, backup bytes, replication factor, request class, refunds, provider cost, protocol fee, treasury amount, and burn amount as separate versioned ledger fields. Every quote must include currency, source, region, tier, `asOf`, tax assumptions, and coverage/confidence where estimated.

Required scenario sheet: free quota; 1/10/100 GiB active user; low/median/high egress; scanner and backup costs; support/abuse reserve; gross-margin candidate; subsidy budget; provider outage and exit egress. User charge must be previewed and approved. Hidden spread, unrealized-profit fees, fake revenue, and guaranteed durability are prohibited.
