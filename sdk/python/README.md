# YNX Chain Python SDK

Dependency-free Python client for YNX Chain REST status, EVM JSON-RPC, and reversible `0x...` / `ynx1...` address conversion. Use `to_ynx_address`, `to_evm_address`, or `normalize_ynx_address`; both representations identify the same 20-byte YNX account, while EVM RPC and MetaMask continue to use `0x...`. The package is prepared for local installation and validation; this repository does not claim that it is published to PyPI.
