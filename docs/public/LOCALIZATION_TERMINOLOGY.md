## Required controlled terms

Translators must distinguish Testnet/Mainnet; candidate/active; locally tested/publicly deployed; signed/unsigned; simulation/actual; submitted/confirmed/failed/unknown; gross/net; fee/cost; reserve/reported reserve; redeemable/not evidenced; custody/non-custodial; revoke/expire; and risk/no guarantee. Do not translate “no represented monetary value” into “free,” “worthless,” or “guaranteed not to change.”

Arabic resources must set `dir="rtl"`, use logical CSS properties, preserve left-to-right rendering for hashes/addresses/IDs, and mirror directional controls only when semantics require it. All locales require plural/date/number/currency handling, screen-reader labels, errors, dynamic states, legal/risk text and fallback tests. Menu-only localization does not pass.
