# YNX DEX 0.1.0 Testnet Preview candidate

Local candidate adds an independent Web/PWA, immutable constant-product contracts, bounded Router/Quoter behavior, JavaScript SDK, owner-reviewed token-list boundary, persistent Indexer/API with confirmed EVM scanning and reorg recovery, canonical Wallet integration manifest, deterministic live-reserve quotes, Pool Detail and liquidity review, AI risk-explanation workflow, twelve locales, Playwright/offline tests and a deterministic upload-ready PWA bundle.

The final branch additionally adds a non-upgradeable Strategy Vault source candidate. It binds an immutable user owner and limited engine, allows only typed swap/liquidity actions returning assets to the vault, enforces exact approvals, nonce/frequency/capital/gas/deadline/oracle/depeg/loss/drawdown limits, and preserves owner pause/revoke/kill/withdraw/emergency exit. Performance fees are disabled because realized-PnL accounting is not yet available.

The JavaScript SDK adds a limited Execution Adapter. It does not contain keys or an automatic strategy loop: submission requires an exact request digest from canonical Wallet introspection and an explicitly injected engine transport, returns an unconfirmed state, then requires confirmed event reconciliation. Source-labelled gas, public fee, owner-reviewed Oracle and risk observations fail closed against the Vault mandate. CPMM LP fees are truthfully reported as embedded in reserves, so the SDK refuses to fabricate a separate collect transaction. Compound is only an explicitly sized add-liquidity request; rebalance stops after its first request until confirmed reconciliation, fresh state and another Wallet approval.

Indexer schema/cursor v2 adds confirmed Strategy Vault `ActionExecuted` ingestion, Vault-address binding, source-labelled reconciliation records and shared pool/Vault reorg recovery. Authenticated schema-v1 state and cursor files migrate only after exact rollback backups are preserved; enabling Vault indexing from a legacy cursor forces a deployment-block rescan.

This is not a deployed release. The observed public RPC identifies YNX Testnet chain 6423, but no verified contract addresses, reviewed tokens, pools, liquidity, Wallet swap/LP proof, staging/public URL, hosted artifact, production signature, store review or independent audit exists. All corresponding central/deployment/distribution flags remain false.
