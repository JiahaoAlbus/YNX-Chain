# YNX Chain Whitepaper

YNX Chain is a Web4 L1 concept and engineering stack centered on assets, identity, AI operations, resources, payments, evidence, reputation, permissions, and automated execution.

YNXT is the native coin, gas token, staking token, resource staking token, and faucet token.

The current repository demonstrates local devnet and public-testnet deployment readiness. Mainnet readiness requires external audit, legal review, validator decentralization, public stress tests, and governance setup.
