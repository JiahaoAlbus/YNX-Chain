# Bridge Unit Economics

No Bridge fee, spread, revenue, subsidy, or profitability claim is active.

Cost accounting must separate source gas, destination gas, provider or attestation cost, relayer/HSM cost, RPC and indexing cost, compute, storage, monitoring, support, dispute handling, security review, and insurance-pool contributions. Each value requires currency, source, timestamp, jurisdiction, confidence, and whether it is paid by YNX, a provider, or the user.

User-visible fees must be quoted before Wallet approval and later reconciled against receipts. Hidden spread, fees on unrealized gains, undisclosed mint/burn economics, and guaranteed returns are prohibited.

Per-active-user cost, free allowance, subsidy budget, gross-margin candidate, and scale/stop decision remain unmeasured because no funded public Testnet route or representative usage dataset exists.
