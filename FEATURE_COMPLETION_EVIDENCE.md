## Current release truth

The complete product is not locally implemented or tested. The event core, local transactional state, JetStream transport adapter, initial PostgreSQL schema/migrator, Saga model, ledger model, reconciliation model, SDK facade, schema files, and fail-closed HTTP boundary are implemented and locally tested at the evidence levels stated above. This narrower component evidence must not be promoted to a product-level release state.
